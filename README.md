# PowerBI-Superstore-profit-report-Report-2026--Dashboard
Power BI Report analyzing Superstore sales and profit.
Superstore Profit Report - Power BI Project

Tools Used
- Power BI
- DAX

Key Metrics
- Total Sales
- Total Profit
- Profit Margin
- Category wise profit analysis

Sample DAX Measures

Total Sales = SUM(Orders[Sales])

Total Profit = SUM(Orders[Profit])

Profit Margin = DIVIDE([Total Profit],[Total Sales],0)
